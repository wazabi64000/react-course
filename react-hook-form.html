React Hook Form

← Retour au menu

React Hook Form est une bibliothèque qui simplifie la gestion des formulaires dans React. Elle permet de récupérer les valeurs, valider les champs et gérer l'état du formulaire facilement.

Les éléments principaux sont :

  • useForm : initialise le formulaire
  • register : connecte un champ au formulaire
  • handleSubmit : récupère les données lors de l'envoi
  • formState : contient les erreurs

Exercice 1 — Initialiser useForm

Découvrir comment initialiser React Hook Form.

Code Exemple


// Import du hook useForm depuis la bibliothèque react-hook-form
import { useForm } from "react-hook-form"

// Création du composant React
function FormExample(){

  // Initialisation du formulaire
  // register : connecte les champs
  // handleSubmit : gère l'envoi du formulaire
  const { register, handleSubmit } = useForm()

  // Fonction appelée lors de l'envoi
  function onSubmit(data){

    // data contient toutes les valeurs du formulaire
    console.log(data)

  }

  return (

    // Formulaire HTML
    // handleSubmit capture les données avant l'envoi
    <form onSubmit={handleSubmit(onSubmit)}>

      // Champ email connecté au formulaire
      <input {...register("email")} />

      // Bouton d'envoi
      <button>Envoyer</button>

    </form>

  )

}

Exercice 3 — handleSubmit

Récupérer les données lors de l'envoi du formulaire. 


// Import du hook
import { useForm } from "react-hook-form"

function LoginForm(){

 // register connecte les inputs
 // handleSubmit gère l'envoi

 // Fonction appelée au submit
 function onSubmit(data){

  // data contient toutes les valeurs
  console.log(data)

 }

 return(

  // Formulaire
  <form onSubmit={handleSubmit(onSubmit)}>

   // Champ email
   <input {...register("email")} />

   // Champ mot de passe

   // Bouton login

  </form>

 )

}

Exercice 8 — Register fictif

Créer un formulaire d'inscription simple. 


import { useForm } from "react-hook-form"

function Register(){

 const { register, handleSubmit } = useForm()

 function onSubmit(data){

  // simulation création utilisateur
  console.log("Utilisateur créé :", data)

 }

 return(

  <form onSubmit={handleSubmit(onSubmit)}>

   // Nom utilisateur
   <input {...register("name")} placeholder="Nom" />

   // Email
   <input {...register("email")} placeholder="Email" />

   // Mot de passe
   <input type="password" {...register("password")} />

   // Bouton création compte
   <button>Créer compte</button>

  </form>

 )

}

Exercice 10 — Formulaire complet

Créer un formulaire avec validation et messages d'erreur. 


import { useForm } from "react-hook-form"

function Contact(){

 const {

  register,

  handleSubmit,

  formState: { errors }

 } = useForm()

 function onSubmit(data){

  console.log(data)

 }

 return(

  <form onSubmit={handleSubmit(onSubmit)}>

   // Champ nom obligatoire
   <input {...register("name",{required:true})} />

   // Message erreur
   {errors.name && <p>Nom requis</p>}

   // Champ email obligatoire
   <input {...register("email",{required:true})} />

   // Message erreur
   {errors.email && <p>Email requis</p>}

   // Bouton envoyer
   <button>Envoyer</button>

  </form>

 )

}
