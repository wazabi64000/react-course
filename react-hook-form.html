React Hook Form

← Retour au menu

React Hook Form est une bibliothèque qui simplifie la gestion des formulaires dans React. Elle permet de récupérer les valeurs des champs, gérer la validation et contrôler l'état du formulaire.

Les éléments principaux sont :

  • useForm : initialise le formulaire
  • register : connecte un input au formulaire
  • handleSubmit : récupère les données lors de l'envoi
  • formState : contient les erreurs et l'état du formulaire
---

Exercice 1 — Initialiser un formulaire

Objectif

Initialiser React Hook Form avec useForm.

Code exemple


import { useForm } from "react-hook-form"

function RegisterForm(){

  const { register, handleSubmit } = useForm()

  function onSubmit(data){
    console.log(data)
  }

  return (
    <form onSubmit={handleSubmit(onSubmit)}>
      <input {...register("email")} />
      <button>Envoyer</button>
    </form>
  )

}

Explication

1. useForm() initialise le formulaire.
2. register connecte un champ au formulaire.
3. handleSubmit récupère les données au submit.

const { register, handleSubmit } = useForm()

Exercice 10 — Mini formulaire complet

Objectif

Créer un formulaire avec validation. 

function Contact(){

 const {
  register,
  handleSubmit,
  formState: { errors }
 } = useForm()

 function onSubmit(data){
  console.log(data)
 }

 return(

  <form onSubmit={handleSubmit(onSubmit)}>

   <input {...register("name",{required:true})} />

   {errors.name && <p>Nom requis</p>}

   <input {...register("email",{required:true})} />

   {errors.email && <p>Email requis</p>}

   <button>Envoyer</button>

  </form>

 )

}
